  5. Wait 10-30 seconds
  6. View 3D scene!

  3. Works with both models

üìä WHAT THE SETUP SCRIPT DOES
================================================================================

1. Creates conda environment 'world3d'
2. Installs PyTorch 2.5.0 with CUDA 12.4
3. Installs Gradio and dependencies
4. Clones HunyuanWorld-1.0 repository
   - Installs Real-ESRGAN (super-resolution)
   - Installs ZIM (depth estimation)
   - Downloads model checkpoints
5. Clones WorldGen repository
   - Installs PyTorch3D
   - Installs dependencies
   - Optional: ML-Sharp experimental features
6. Configures Hugging Face authentication
7. Downloads FLUX.1-dev base model (~10GB)

Total download: ~20GB
Total time: 30-60 minutes

üîß TROUBLESHOOTING QUICK REFERENCE
================================================================================

"CUDA out of memory"
  ‚Üí Close other GPU apps
  ‚Üí Use one model at a time
  ‚Üí Quantization already enabled

"Model not found"
  ‚Üí Run: huggingface-cli login
  ‚Üí Accept FLUX.1-dev license

"Port 7860 in use"
  ‚Üí Run: lsof -i :7860
  ‚Üí Kill process or use different port

"Setup fails"
  ‚Üí Check internet connection
  ‚Üí Ensure 50GB+ free space
  ‚Üí Verify CUDA drivers installed

"Slow generation"
  ‚Üí First run is always slower
  ‚Üí Subsequent runs are faster
  ‚Üí Enable caching (see docs)
